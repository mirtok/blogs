Linux(Centos7)安装jdk1.8

Li Xiang 2021-10-15 LinuxCentos7JDK1.8

# Linux安装jdk(两种方式)

# 1. 不管那一步都要查看系统是否自带jdk

# 卸载系统自带的OpenJDK以及相关的java文件 
# java -version 结果 存在
# 查看相关java文件 rpm -qa | grep java
# 删除相关文件
  rpm -e --nodeps java-1.8.0-openjdk-headless-1.8.0.65-3.b17.el7.x86_64
  rpm -e --nodeps java-1.8.0-openjdk-1.8.0.65-3.b17.el7.x86_64
# 查看删除结果
  java -version

# yum 安装jdk1.8

#查询 jdk
    yum -y list java*
  1. 查询 jdk yum -y list java*
  2. 安装jdk1.8
  3. yum install -y java-1.8.0-openjdk.x86_64
  4. java -version
  5. 这样就安装成功了。默认给安装到usr/lib/jvm/

# 手动安装

  1. 先去官网下载jdk1.8
  2. 点我快速进入官网 (opens new window)
  3. 没有账号需要注册一下oracle账号。
  4. 第一次注册需要邮箱激活一下。
  5. 然后就可以用邮箱+密码直接登录。 登录完之后可以直接下载！
  6. 服务器创建文件夹
#进入根目录
cd / 
#在根目录创建jdk目录
mkdir jdk
  1. 上传服务器
  2. 上传完成：在jdk目录下ll查询一下
  3. 解压jdk：
  4. 命令tar -zxvf jdk-8u211-linux-x64.tar.gz（jdk-8..这个是刚下载的jdk名称）
  5. 解压成功
  6. 配置环境变量 vim /etc/profile
  7. 配置一下环境变量代码： 
#将以下命令放到环境变量 注意文件夹名称
export JAVA_HOME=/DATA/jdk/jdk1.8.0_211 
export CLASSPATH=$:CLASSPATH:$JAVA_HOME/lib/ 
export PATH=$PATH:$JAVA_HOME/bin
#点击esc 进入命令模式 输入：wq! 保存修改信息

14. 刷新环境变量文件： 刷新环境变量命令:source /etc/profile 15. 查看是否安装成功: 查询jdk版本命令： java -version 16. 关于如果没有oracle账号的效果版可以看这里 点我获取账号哦! (opens new window)
